# Munch It Personality Quiz

A full-stack application to determine your snack personality based on 7 fun questions.

## Setup Instructions

### Frontend
1. Open a terminal and navigate to `frontend`
2. Run `npm install`
3. Run `npm run dev`

### Backend
1. Open a terminal and navigate to `backend`
2. Run `npm install`
3. Run `npm run dev`

The frontend communicates with the backend on port 5000.
